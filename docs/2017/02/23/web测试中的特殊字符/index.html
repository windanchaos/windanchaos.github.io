数据库特殊字符：

_ (下划线) : Oracle中代表占位符
表示查找含_的字符串。
%（百分号）表示查找含%的字符串。
‘(单引号)：表示查找含’的字符串
/#井号表示查找含/#的字符串
通配符，和需求相关。如果/*和?当作通配符处理，则不需要数据库开发人员特殊处理。如果当作其本身这个字符，则需要处理。所以需要测试是否已经做了正确的处理。
/*（星号）：代表通配任意多个字符或数字进行查询
?（问号）：代表通配1个字符或数字进行查询
